# Calculator Project (Python)

This is a simple console-based Calculator created as part of Assignment 4 for the course **Computer Science Fundamentals & Career Pathways (ETCCCP105)**.

The calculator performs the four basic arithmetic operations:
- Addition  
- Subtraction  
- Multiplication  
- Division  

The purpose of this project is to learn:
- How to develop a mini project in Python
- How to use Git for version control
- How to maintain a clean and professional GitHub repository
- How to document a project using Markdown
- How to use VS Code and GitHub effectively
