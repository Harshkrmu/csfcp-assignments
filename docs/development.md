# Development Process

1. Set up the project folder inside VS Code  
2. Created calculator.py with modular functions  
3. Initialized Git using `git init`  
4. Created commits for every major change  
5. Added README.md and documentation folder  
6. Created a GitHub repository  
7. Linked local repo to GitHub using `git remote add origin`  
8. Pushed all changes using `git push`  
9. Updated project with screenshots and improvements
